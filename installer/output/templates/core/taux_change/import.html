Format du fichier CSV

Le fichier doit contenir les colonnes suivantes (dans cet ordre) :

  • devise_source : Code de la devise source (ex: USD, EUR)
  • devise_cible : Code de la devise cible (ex: GNF)
  • taux : Taux de change (nombre décimal)
  • date_taux : Date au format YYYY-MM-DD
  • source (optionnel) : Source du taux (ex: BCRG)

La première ligne doit contenir les noms des colonnes.

Exemple de fichier CSV
devise_source,devise_cible,taux,date_taux,source
USD,GNF,8600.00,2024-01-15,BCRG
EUR,GNF,9500.00,2024-01-15,BCRG
USD,EUR,0.905263,2024-01-15,Marché international
XOF,GNF,14.50,2024-01-15,BCRG
Notes importantes
  • Les devises doivent exister dans le système avant l'import
  • Si un taux existe déjà pour une combinaison devise/date, il sera mis à jour
  • Les erreurs d'import seront affichées ligne par ligne
  • L'encodage du fichier doit être UTF-8
