📋 PRÉREQUIS
------------
✅ Python 3.10+ installé
✅ PostgreSQL 14+ installé
✅ Git installé (optionnel)

================================================================================
⚡ INSTALLATION EN 8 ÉTAPES (15 minutes)
================================================================================

ÉTAPE 1 : Ouvrir PowerShell
----------------------------
Windows + R → tapez "powershell" → Entrée

ÉTAPE 2 : Aller dans le dossier du projet
------------------------------------------
cd C:\Users\LENO\Desktop\GestionnaireRH

ÉTAPE 3 : Créer l'environnement virtuel
----------------------------------------
python -m venv venv

ÉTAPE 4 : Activer l'environnement virtuel
------------------------------------------
.\venv\Scripts\activate

Vous devriez voir (venv) au début de la ligne

ÉTAPE 5 : Installer les dépendances
------------------------------------
pip install -r requirements.txt

⏱️ Temps : ~2-3 minutes

ÉTAPE 6 : Créer la base de données
-----------------------------------
cd database
.\install_database.bat

Suivre les instructions à l'écran
Mot de passe PostgreSQL par défaut : postgres

⏱️ Temps : ~1 minute

ÉTAPE 7 : Appliquer les migrations Django
------------------------------------------
cd ..
python manage.py migrate

ÉTAPE 8 : Créer un compte administrateur
-----------------------------------------
python manage.py createsuperuser

Remplir :
- Username: admin
- Email: admin@example.com
- Password: (votre mot de passe sécurisé)
- Password (again): (répéter le mot de passe)

================================================================================
🎯 LANCER L'APPLICATION
================================================================================

1. Activer l'environnement (si pas déjà fait)
   .\venv\Scripts\activate

2. Lancer le serveur
   python manage.py runserver

3. Ouvrir le navigateur
   http://localhost:8000

4. Se connecter
   Username: admin
   Password: (celui créé à l'étape 8)

================================================================================
❓ PROBLÈMES COURANTS
================================================================================

PROBLÈME : "Module not found"
SOLUTION : pip install -r requirements.txt

PROBLÈME : "Connection refused" (PostgreSQL)
SOLUTION : Vérifier que PostgreSQL est démarré
   Services Windows > PostgreSQL > Démarrer

PROBLÈME : "Port already in use"
SOLUTION : Changer de port
   python manage.py runserver 8080

PROBLÈME : Page blanche / erreur 500
SOLUTION : Vérifier les logs dans la console

PROBLÈME : Fichiers statiques non chargés
SOLUTION : python manage.py collectstatic
