🎯 OBJECTIF
───────────
Déclarer et payer les charges sociales mensuelles (CNSS, IRG, INAM)


📍 ACCÈS
────────
URL : http://127.0.0.1:8000/paie/declarations/
Navigation : Sidebar → RAPPORTS → Déclarations


✅ ÉTAPES (5 MINUTES)
─────────────────────

ÉTAPE 1 : CALCULER LES BULLETINS
─────────────────────────────────
1. Aller sur /paie/periodes/
2. Sélectionner la période (ex: Octobre 2025)
3. Cliquer sur "Calculer la Période"
4. ✅ Tous les bulletins sont générés


ÉTAPE 2 : ACCÉDER AUX DÉCLARATIONS
───────────────────────────────────
1. Aller sur /paie/declarations/
2. Sélectionner :
   - Année : 2025
   - Mois : 10 (Octobre)
3. Cliquer sur "Filtrer"


ÉTAPE 3 : VÉRIFIER LES MONTANTS
────────────────────────────────
La page affiche automatiquement :

┌─────────────────────────────────────────┐
│ 📊 CNSS                                 │
│ • Masse salariale : 112,500,000 GNF     │
│ • Cotisation employé (5%) : 5,625,000   │
│ • Cotisation employeur (18%) : 20,250   │
│ • TOTAL : 25,875,000 GNF                │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 💰 IRG                                  │
│ • Masse imposable : 112,500,000 GNF     │
│ • IRG retenu : 8,437,500 GNF            │
│ • TOTAL : 8,437,500 GNF                 │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│ 🏥 INAM                                 │
│ • Masse salariale : 112,500,000 GNF     │
│ • Taux : 2.5%                           │
│ • TOTAL : 2,812,500 GNF                 │
└─────────────────────────────────────────┘


ÉTAPE 4 : EXPORTER EN PDF
──────────────────────────
1. Cliquer sur "Imprimer" (en haut à droite)
2. Sélectionner "Enregistrer au format PDF"
3. Nommer : Declaration_Octobre_2025.pdf
4. Enregistrer


ÉTAPE 5 : PAYER ET SOUMETTRE
─────────────────────────────

📅 10 NOVEMBRE (IRG)
────────────────────
• Montant : 8,437,500 GNF
• Où : Trésor Public / Centre des Impôts
• Documents : PDF + Liste employés

📅 15 NOVEMBRE (CNSS)
─────────────────────
• Montant : 25,875,000 GNF
• Où : Agence CNSS
• Documents : PDF + Bordereau

📅 15 NOVEMBRE (INAM)
─────────────────────
• Montant : 2,812,500 GNF
• Où : Agence INAM
• Documents : PDF
